# Getting Started with Fractional Calculator

## Available Scripts

In the project directory, you can run:

### `npm start`

Operations: add +
Operations: subtract -
Operations: multiply +
Operations: divide -

Example run:
? 1/2 * 3_3/4
= 1_7/8 

?2_3/8 + 9/8
= 3_1/2
